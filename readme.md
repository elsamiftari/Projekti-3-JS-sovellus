Tietoja sovelluksesta
- Kissa- ja koirakuvien selaaja on sovellus, joka näyttää satunnaisen kissan tai koiran kuvan napin painalluksella. 


Tunnetut virheet/bugit

Teknologiat
Käytin seuraavia teknologioita:  
- HTML & CSS – rakenne ja perusulkoasu  
- Bootstrap – napit, kortti ja responsiivisuus  
- jQuery – napin kuuntelu, DOM-manipulaatio  
- Axios – API-kutsujen tekeminen (kissan ja koiran kuvat)

Asennus
1. Lataa tai kloonaa repositorio koneellesi.  
2. Avaa `index.html` selaimessa.  
3. Napin painallus hakee satunnaisen kuvan API:sta. 


Kiitokset
Käytin apuna kurssin materiaaleja, eri opetusvideoita, ChatGpt:tä käytin asioiden selittämiseen eli jos en tajunnu jotain ja miten joku asia menee pysin sen opettamaan ja selittäämään sen minulle.
